# Project Brief

## Site: cverdin.com (rawrdoge.github.io)

### Purpose
Personal social hub landing page with glassmorphism design language.

### Core Design Principles
- Glassmorphism UI: frosted glass effect with `backdrop-filter: blur(12px)`
- Gradient backgrounds: Cross-browser (`-moz-`, `-webkit-`, standard)
- Adaptive viewport: 70% width nav container, responsive breakpoints
- Centered logo prominence, horizontal social links below
